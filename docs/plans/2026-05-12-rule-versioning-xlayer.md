# S31 规则版本化与 X Layer 接入

## 背景

OKX Boost 当前公开规则已变化，旧代码里的 `0.85` 不能继续作为新交易默认基础倍数。同时，用户需要把 X Layer 上通过 OKX DEX Router 的交易合并进同一个钱包的 10 日 Boost 结果。

## 实现边界

1. 规则按交易时间选择版本，不按扫描时间整体重算。
2. 2026-05-12 00:00:00 UTC 前的交易保留旧规则。
3. 2026-05-12 00:00:00 UTC 起的交易使用新规则。
4. 单钱包扫描结果合并 BNB Chain 和 X Layer。
5. X Layer 交易发现走 Ankr Advanced `xlayer` 钱包索引，不使用公共 RPC 扫 10 天 Transfer 日志兜底。

## 已完成

1. 新增规则版本字段与缓存版本，避免新旧规则缓存串用。
2. 新规则基础倍数：类别 1/2 与其他代币为 `0.5x`，类别 1 与类别 1 为 `0.1x`，类别 1/2 内部相关组合为 `0.25x`。
3. X Layer 自动叠加 `1.2x` 链级额外加成。
4. 新增 X Layer 链配置、OKX Router、USDt0 稳定币识别和交易浏览器链接。
5. 前端详情页交易明细展示链名，并用对应链的浏览器链接。
6. Cron 自动刷新改为多链合并。
7. 本地构建和真实链上演练已通过。

## 验收

1. `npm run lint` 通过。
2. `npm run build` 通过。
3. 用钱包 `0x35217ad88c31db4c95e67b77e68795ea4d54cc30`、快照日 `2026-05-12` 真实演练成功。
4. 用户给出的两笔 X Layer 交易均被识别为 `USDt0 <-> QIC`，按 `0.5 × 1.2` 计算。
