# S35 禁用私有访问码归档回退

## 目标

避免用户以为正在读取 Supabase 账号归档，实际却因为未登录或登录态过期而回退到私有访问码的数据空间。云端归档只使用 Supabase 登录账号。

## 实现

1. 前端归档读取和写入只在存在 Supabase session 时执行。
2. 未登录时不再用私有访问码读取 Upstash 数据空间归档。
3. 私有访问码保留为未登录扫描 API 的访问凭证，不再承担云端归档语义。
4. 钱包管理页文案明确显示未登录时归档状态为“未登录”。

## 验收

1. 未登录时不会请求旧数据空间归档。
2. 已登录时 `/api/archive` 使用 Supabase Bearer token。
3. 私有访问码不会覆盖或混淆 Supabase 账号归档。
