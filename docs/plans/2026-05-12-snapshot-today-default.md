# S32 快照日期实时默认值修复

## 目标

刷新网页后，快照日期必须回到当前 UTC 日，用于实时判断今天的 10 日窗口；历史日期只能由用户在当前会话里主动选择。

## 实现

1. 前端初始化不再读取本地持久化的 `endDate`。
2. 前端保存 UI 状态时不再写入 `endDate`。
3. 读取 Supabase 云端归档时保留当前页面快照日，不让旧归档设置覆盖。
4. Supabase 归档读取优先按当前 UTC 日匹配扫描结果，找不到再回落到最近结果。

## 验收

1. 刷新页面后，工具条快照日显示当前 UTC 日。
2. 云端归档里存在旧日期时，不会把页面快照日改回旧日期。
3. 用户仍可在当前页面手动选择历史快照日。
