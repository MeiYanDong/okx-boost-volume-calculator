# S10 子 Plan：钱包管理独立页面

## 阶段目标

把钱包管理从全局浮层、顶部更多菜单和偏好设置页中移出，收敛成侧边栏独立页面。其他页面只展示交易、扫描、统计和达标结果，不再承担钱包配置入口。

## 范围

1. 新增侧边栏「钱包管理」页面。
2. 钱包名称、地址和私有访问码只在「钱包管理」页面维护。
3. 顶部更多菜单移除钱包入口。
4. 偏好设置页移除钱包列表编辑入口。
5. 删除旧的钱包浮层状态与组件。
6. 无钱包时主动作跳转到「钱包管理」页面，而不是打开浮层。

## 不做

1. 不改变钱包表格作为交易结果展示的定位。
2. 不改扫描、归档、增量刷新、代币加成和计算逻辑。
3. 不引入新的钱包数据库，仍使用本机浏览器中的钱包文本作为来源。

## 验收标准

1. 侧边栏有且只有一个「钱包管理」入口。
2. 顶部更多菜单不再出现钱包管理或钱包列表按钮。
3. 偏好设置页不再出现编辑钱包列表按钮。
4. 页面中不再渲染旧的钱包编辑浮层。
5. `npm run lint` 与 `npm run build` 通过。
6. 浏览器验证钱包管理页可打开，其他页面没有重复钱包编辑入口。
