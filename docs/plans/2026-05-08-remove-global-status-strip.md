# S8 子 Plan：删除全局状态条

## 阶段目标

删除所有页面重复出现的全局状态条，尤其是默认文案「准备同步钱包归档」，让页面只保留当前任务真正需要的信息。

## 范围

1. 删除全局 `StatusStrip` 渲染。
2. 删除 `StatusStrip` 组件和 `.system-strip` 样式。
3. 删除只服务于全局状态条的 `progress` / `error` 状态。
4. 保留钱包行级别的扫描进度和错误状态。

## 不做

1. 不改钱包表格内的状态列。
2. 不改扫描、归档、增量刷新和加成计算逻辑。
3. 不新增新的全局提示区域。

## 验收标准

1. 任何页面都不再出现「准备同步钱包归档」。
2. 页面 DOM 中不再存在 `system-strip`。
3. 扫描时钱包行仍能显示自己的进度。
4. `npm run lint` 与 `npm run build` 通过。
