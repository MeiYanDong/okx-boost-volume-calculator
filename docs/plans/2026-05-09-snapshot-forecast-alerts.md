# S13 快照预警与未来 3 天预测

## 目标

在首页直接回答「当前归档在接下来几次 UTC 日结快照是否仍达标」。计算口径为：快照日包含当天，10 日窗口为快照日及前 9 个 UTC 日期。北京时间每日 08:00 确认上一 UTC 日结果。

## 实现边界

- 基于浏览器本地已归档的 `dailyRows` 计算，不重新扫链。
- 首页新增「快照预警」卡片，显示当前快照和未来 3 次快照。
- 每个预测行展示快照日、北京时间确认时间、10 日窗口、到期日期、合计 Boost、达标钱包数和风险钱包。
- 风险钱包可点击进入钱包详情。
- 切换快照日期时，旧日期的临时 fresh 结果不能继续沿用。

## 验收标准

- 当前快照包含当天。
- 未来快照按窗口滚动时，窗口外日期交易量会被扣除。
- 如果某个钱包未来低于单钱包 10 日累计目标，首页明确提示最早风险时间和最大差额。
- 无已归档钱包时不误报「未来 3 天安全」。
- `npm run build` 通过。
